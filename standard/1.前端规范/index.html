    BEM规范

    BEM（Block、Element、Modifier）是一种命名约定，用于描述组件的结构和状态。

    // Block: 定义功能模块
.form-container {
  // Element: 使用 __ 连接
  &__item {
    padding: 12px;
  }

  // Modifier: 使用 -- 表示状态
  &__item--disabled {
    opacity: 0.5;
  }

  // 子元素继续使用 BEM
  &__label {
    font-weight: bold;
  }
}
