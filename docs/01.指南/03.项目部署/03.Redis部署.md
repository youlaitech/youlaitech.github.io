## 1. 镜像搜索

```bash
docker search redis
```

## 2. 镜像下载

```bash
docker pull redis:latest
```

## 3. 创建目录

``` bash
mkdir -p /mnt/redis/data
```

## 4. 启动容器

```bash
docker run -d --name=redis  \
-p 6379:6379 \
-v /mnt/redis/redis.conf:/etc/redis/redis.conf \
-v /mnt/redis/data:/data \
--restart=always \
redis:latest --requirepass "123456" --appendonly yes
```
