            项目介绍

            vue3-element-admin 是基于 Vue3 + Vite5+ TypeScript5 + Element-Plus + Pinia 等主流技术栈构建的免费开源的中后台管理的前端模板（配套 Java 后端 源码）。

            在线预览： https://vue3.youlai.tech/

            提示

            • 如果你对 vue3-element-admin 框架完全陌生，请从我们的 快速上手 开始。

            • 如果你熟悉上述内容，请从这里开始。

            项目特色

            • 简洁易用：基于 vue-element-admin 升级的 Vue3 版本，无过渡封装 ，易上手。

            • 数据交互：同时支持本地 Mock 和线上接口，配套 Java 后端源码和在线接口文档。

            • 权限管理：用户、角色、菜单、字典、部门等完善的权限系统功能。

            • 基础设施：动态路由、按钮权限、国际化、代码规范、Git 提交规范、常 用组件封装。

            • 持续更新：项目持续开源更新，实时更新工具和依赖。

            寻求帮助

            如果你在使用过程中遇到了 bug，可以 提一个 issue。 如果你遇到了使用问题，可以 开一个讨论 或者 加入 QQ 群 进行反馈。
