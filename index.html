    开发规范加入我们

    vue3-element-admin

    前端框架：基于 Vue3、Vite、TypeScript、Element-Plus 构建的高效极简企业级后台管理前端模板。

    youlai-boot

    Java后端框架：基于 Java、Spring Boot 3、Spring Security 构建的安全可靠、简洁易扩展的 RBAC 权限管理后端。

    vue-uniapp-template

    移动端框架：基于 Uni-App、Vue3 和 TypeScript 构建的灵活跨平台移动端模板，使用 VSCode 和 CLI 高效开发模式。

    youlai-nest

    Node后端框架：基于 Node、Nest 和 TypeScript 构建的 RBAC 权限管理后端，前端全栈开发首选。

    youlai-cloud

    微服务框架：基于 Java、Spring Boot 3、Spring Cloud & Alibaba、SAS 构建的微服务解决方案。

    youlai-mall

    微服务商城：基于 vue3-element-admin、youlai-cloud、uniapp-nutui-template 构建的微服务商城

    mall-admin

    商城管理前端：基于 vue3-element-admin 模板框架开发的 youlai-mall 商城管理系统

    mall-app

    商城移动端：youlai-mall 微信小程序/H5/Android/iOS 移动应用端。
