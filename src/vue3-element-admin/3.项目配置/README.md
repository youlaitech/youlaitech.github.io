## 修改项目名称

要更改项目名称，需要更新两个文件：`index.html` 和 `package.json`。

- **更新 `index.html`**

在 `src/index.html` 文件中，找到 `<title>` 标签，并将其内容替换为新的项目名称。

- **更新 `package.json`**

在 `package.json` 文件中，找到 `name` 字段，并将其值替换为新的项目名称。
